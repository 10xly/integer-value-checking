# is-eq-negative-number-eighteen
Equality checker for the number -18.

## Installation
```bash
npm install is-eq-negative-number-eighteen
```

## Usage
```js
const iseighteen = require("is-eq-negative-number-eighteen")
const assert = require("assert-fn")

assert(iseighteen(-18))
```
