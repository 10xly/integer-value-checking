# is-eq-positive-number-ninety-three
Equality checker for the number 93.

## Installation
```bash
npm install is-eq-positive-number-ninety-three
```

## Usage
```js
const isninetythree = require("is-eq-positive-number-ninety-three")
const assert = require("assert-fn")

assert(isninetythree(93))
```
