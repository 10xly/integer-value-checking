# is-eq-positive-number-sixty-eight
Equality checker for the number 68.

## Installation
```bash
npm install is-eq-positive-number-sixty-eight
```

## Usage
```js
const issixtyeight = require("is-eq-positive-number-sixty-eight")
const assert = require("assert-fn")

assert(issixtyeight(68))
```
