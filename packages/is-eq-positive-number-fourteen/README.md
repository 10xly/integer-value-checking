# is-eq-positive-number-fourteen
Equality checker for the number 14.

## Installation
```bash
npm install is-eq-positive-number-fourteen
```

## Usage
```js
const isfourteen = require("is-eq-positive-number-fourteen")
const assert = require("assert-fn")

assert(isfourteen(14))
```
