# is-eq-positive-number-sixty-six
Equality checker for the number 66.

## Installation
```bash
npm install is-eq-positive-number-sixty-six
```

## Usage
```js
const issixtysix = require("is-eq-positive-number-sixty-six")
const assert = require("assert-fn")

assert(issixtysix(66))
```
