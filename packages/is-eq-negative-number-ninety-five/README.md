# is-eq-negative-number-ninety-five
Equality checker for the number -95.

## Installation
```bash
npm install is-eq-negative-number-ninety-five
```

## Usage
```js
const isninetyfive = require("is-eq-negative-number-ninety-five")
const assert = require("assert-fn")

assert(isninetyfive(-95))
```
