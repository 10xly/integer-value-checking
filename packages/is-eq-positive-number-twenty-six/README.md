# is-eq-positive-number-twenty-six
Equality checker for the number 26.

## Installation
```bash
npm install is-eq-positive-number-twenty-six
```

## Usage
```js
const istwentysix = require("is-eq-positive-number-twenty-six")
const assert = require("assert-fn")

assert(istwentysix(26))
```
