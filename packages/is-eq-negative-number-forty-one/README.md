# is-eq-negative-number-forty-one
Equality checker for the number -41.

## Installation
```bash
npm install is-eq-negative-number-forty-one
```

## Usage
```js
const isfortyone = require("is-eq-negative-number-forty-one")
const assert = require("assert-fn")

assert(isfortyone(-41))
```
