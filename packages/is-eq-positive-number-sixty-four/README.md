# is-eq-positive-number-sixty-four
Equality checker for the number 64.

## Installation
```bash
npm install is-eq-positive-number-sixty-four
```

## Usage
```js
const issixtyfour = require("is-eq-positive-number-sixty-four")
const assert = require("assert-fn")

assert(issixtyfour(64))
```
