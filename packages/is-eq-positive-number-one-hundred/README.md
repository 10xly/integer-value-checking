# is-eq-positive-number-one-hundred
Equality checker for the number 100.

## Installation
```bash
npm install is-eq-positive-number-one-hundred
```

## Usage
```js
const isonehundred = require("is-eq-positive-number-one-hundred")
const assert = require("assert-fn")

assert(isonehundred(100))
```
