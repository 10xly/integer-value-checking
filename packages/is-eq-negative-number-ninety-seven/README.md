# is-eq-negative-number-ninety-seven
Equality checker for the number -97.

## Installation
```bash
npm install is-eq-negative-number-ninety-seven
```

## Usage
```js
const isninetyseven = require("is-eq-negative-number-ninety-seven")
const assert = require("assert-fn")

assert(isninetyseven(-97))
```
