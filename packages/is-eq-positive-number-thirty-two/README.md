# is-eq-positive-number-thirty-two
Equality checker for the number 32.

## Installation
```bash
npm install is-eq-positive-number-thirty-two
```

## Usage
```js
const isthirtytwo = require("is-eq-positive-number-thirty-two")
const assert = require("assert-fn")

assert(isthirtytwo(32))
```
