# is-eq-positive-number-eighteen
Equality checker for the number 18.

```bash
npm install is-eq-positive-number-eighteen
```

```js
const iseighteen = require("is-eq-positive-number-eighteen")
const assert = require("assert-fn")

assert(iseighteen(18))
```
