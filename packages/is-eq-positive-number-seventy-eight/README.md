# is-eq-positive-number-seventy-eight
Equality checker for the number 78.

## Installation
```bash
npm install is-eq-positive-number-seventy-eight
```

## Usage
```js
const isseventyeight = require("is-eq-positive-number-seventy-eight")
const assert = require("assert-fn")

assert(isseventyeight(78))
```
