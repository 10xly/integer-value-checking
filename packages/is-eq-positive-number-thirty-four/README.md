# is-eq-positive-number-thirty-four
Equality checker for the number 34.

## Installation
```bash
npm install is-eq-positive-number-thirty-four
```

## Usage
```js
const isthirtyfour = require("is-eq-positive-number-thirty-four")
const assert = require("assert-fn")

assert(isthirtyfour(34))
```
