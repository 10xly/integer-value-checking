# is-eq-positive-number-fifty-six
Equality checker for the number 56.

## Installation
```bash
npm install is-eq-positive-number-fifty-six
```

## Usage
```js
const isfiftysix = require("is-eq-positive-number-fifty-six")
const assert = require("assert-fn")

assert(isfiftysix(56))
```
