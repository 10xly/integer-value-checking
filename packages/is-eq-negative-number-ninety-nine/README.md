# is-eq-negative-number-ninety-nine
Equality checker for the number -99.

## Installation
```bash
npm install is-eq-negative-number-ninety-nine
```

## Usage
```js
const isninetynine = require("is-eq-negative-number-ninety-nine")
const assert = require("assert-fn")

assert(isninetynine(-99))
```
