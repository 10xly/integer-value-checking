# is-eq-negative-number-seventy-five
Equality checker for the number -75.

## Installation
```bash
npm install is-eq-negative-number-seventy-five
```

## Usage
```js
const isseventyfive = require("is-eq-negative-number-seventy-five")
const assert = require("assert-fn")

assert(isseventyfive(-75))
```
