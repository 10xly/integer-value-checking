# is-eq-positive-number-twenty-two
Equality checker for the number 22.

## Installation
```bash
npm install is-eq-positive-number-twenty-two
```

## Usage
```js
const istwentytwo = require("is-eq-positive-number-twenty-two")
const assert = require("assert-fn")

assert(istwentytwo(22))
```
