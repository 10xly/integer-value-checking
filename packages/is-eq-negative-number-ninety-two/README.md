# is-eq-negative-number-ninety-two
Equality checker for the number -92.

## Installation
```bash
npm install is-eq-negative-number-ninety-two
```

## Usage
```js
const isninetytwo = require("is-eq-negative-number-ninety-two")
const assert = require("assert-fn")

assert(isninetytwo(-92))
```
