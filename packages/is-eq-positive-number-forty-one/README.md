# is-eq-positive-number-forty-one
Equality checker for the number 41.

## Installation
```bash
npm install is-eq-positive-number-forty-one
```

## Usage
```js
const isfortyone = require("is-eq-positive-number-forty-one")
const assert = require("assert-fn")

assert(isfortyone(41))
```
