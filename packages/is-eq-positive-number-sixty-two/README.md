# is-eq-positive-number-sixty-two
Equality checker for the number 62.

## Installation
```bash
npm install is-eq-positive-number-sixty-two
```

## Usage
```js
const issixtytwo = require("is-eq-positive-number-sixty-two")
const assert = require("assert-fn")

assert(issixtytwo(62))
```
