# is-eq-negative-number-fifty-seven
Equality checker for the number -57.

## Installation
```bash
npm install is-eq-negative-number-fifty-seven
```

## Usage
```js
const isfiftyseven = require("is-eq-negative-number-fifty-seven")
const assert = require("assert-fn")

assert(isfiftyseven(-57))
```
