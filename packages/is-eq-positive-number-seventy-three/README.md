# is-eq-positive-number-seventy-three
Equality checker for the number 73.

## Installation
```bash
npm install is-eq-positive-number-seventy-three
```

## Usage
```js
const isseventythree = require("is-eq-positive-number-seventy-three")
const assert = require("assert-fn")

assert(isseventythree(73))
```
