# is-eq-negative-number-eighty-two
Equality checker for the number -82.

## Installation
```bash
npm install is-eq-negative-number-eighty-two
```

## Usage
```js
const iseightytwo = require("is-eq-negative-number-eighty-two")
const assert = require("assert-fn")

assert(iseightytwo(-82))
```
