# is-eq-positive-number-ninety-seven
Equality checker for the number 97.

## Installation
```bash
npm install is-eq-positive-number-ninety-seven
```

## Usage
```js
const isninetyseven = require("is-eq-positive-number-ninety-seven")
const assert = require("assert-fn")

assert(isninetyseven(97))
```
