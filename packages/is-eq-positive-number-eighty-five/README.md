# is-eq-positive-number-eighty-five
Equality checker for the number 85.

## Installation
```bash
npm install is-eq-positive-number-eighty-five
```

## Usage
```js
const iseightyfive = require("is-eq-positive-number-eighty-five")
const assert = require("assert-fn")

assert(iseightyfive(85))
```
