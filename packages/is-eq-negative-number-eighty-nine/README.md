# is-eq-negative-number-eighty-nine
Equality checker for the number -89.

## Installation
```bash
npm install is-eq-negative-number-eighty-nine
```

## Usage
```js
const iseightynine = require("is-eq-negative-number-eighty-nine")
const assert = require("assert-fn")

assert(iseightynine(-89))
```
