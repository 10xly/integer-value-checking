# is-eq-positive-number-thirty-seven
Equality checker for the number 37.

## Installation
```bash
npm install is-eq-positive-number-thirty-seven
```

## Usage
```js
const isthirtyseven = require("is-eq-positive-number-thirty-seven")
const assert = require("assert-fn")

assert(isthirtyseven(37))
```
