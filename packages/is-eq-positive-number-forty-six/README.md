# is-eq-positive-number-forty-six
Equality checker for the number 46.

## Installation
```bash
npm install is-eq-positive-number-forty-six
```

## Usage
```js
const isfortysix = require("is-eq-positive-number-forty-six")
const assert = require("assert-fn")

assert(isfortysix(46))
```
