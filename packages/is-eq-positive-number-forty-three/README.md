# is-eq-positive-number-forty-three
Equality checker for the number 43.

## Installation
```bash
npm install is-eq-positive-number-forty-three
```

## Usage
```js
const isfortythree = require("is-eq-positive-number-forty-three")
const assert = require("assert-fn")

assert(isfortythree(43))
```
