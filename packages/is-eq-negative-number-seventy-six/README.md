# is-eq-negative-number-seventy-six
Equality checker for the number -76.

## Installation
```bash
npm install is-eq-negative-number-seventy-six
```

## Usage
```js
const isseventysix = require("is-eq-negative-number-seventy-six")
const assert = require("assert-fn")

assert(isseventysix(-76))
```
