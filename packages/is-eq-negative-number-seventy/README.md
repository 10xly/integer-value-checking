# is-eq-negative-number-seventy
Equality checker for the number -70.

## Installation
```bash
npm install is-eq-negative-number-seventy
```

## Usage
```js
const isseventy = require("is-eq-negative-number-seventy")
const assert = require("assert-fn")

assert(isseventy(-70))
```
