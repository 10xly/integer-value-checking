# is-eq-negative-number-fifty-eight
Equality checker for the number -58.

## Installation
```bash
npm install is-eq-negative-number-fifty-eight
```

## Usage
```js
const isfiftyeight = require("is-eq-negative-number-fifty-eight")
const assert = require("assert-fn")

assert(isfiftyeight(-58))
```
