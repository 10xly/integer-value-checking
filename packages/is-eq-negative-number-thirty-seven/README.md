# is-eq-negative-number-thirty-seven
Equality checker for the number -37.

## Installation
```bash
npm install is-eq-negative-number-thirty-seven
```

## Usage
```js
const isthirtyseven = require("is-eq-negative-number-thirty-seven")
const assert = require("assert-fn")

assert(isthirtyseven(-37))
```
