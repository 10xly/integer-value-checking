# is-eq-positive-number-fifty-eight
Equality checker for the number 58.

## Installation
```bash
npm install is-eq-positive-number-fifty-eight
```

## Usage
```js
const isfiftyeight = require("is-eq-positive-number-fifty-eight")
const assert = require("assert-fn")

assert(isfiftyeight(58))
```
