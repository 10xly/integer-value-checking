# is-eq-positive-number-nineteen
Equality checker for the number 19.

## Installation
```bash
npm install is-eq-positive-number-nineteen
```

## Usage
```js
const isnineteen = require("is-eq-positive-number-nineteen")
const assert = require("assert-fn")

assert(isnineteen(19))
```
