# is-eq-negative-number-twenty-two
Equality checker for the number -22.

## Installation
```bash
npm install is-eq-negative-number-twenty-two
```

## Usage
```js
const istwentytwo = require("is-eq-negative-number-twenty-two")
const assert = require("assert-fn")

assert(istwentytwo(-22))
```
