# is-eq-negative-number-sixteen
Equality checker for the number -16.

## Installation
```bash
npm install is-eq-negative-number-sixteen
```

## Usage
```js
const issixteen = require("is-eq-negative-number-sixteen")
const assert = require("assert-fn")

assert(issixteen(-16))
```
