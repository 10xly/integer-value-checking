# is-eq-positive-number-seventy-seven
Equality checker for the number 77.

## Installation
```bash
npm install is-eq-positive-number-seventy-seven
```

## Usage
```js
const isseventyseven = require("is-eq-positive-number-seventy-seven")
const assert = require("assert-fn")

assert(isseventyseven(77))
```
