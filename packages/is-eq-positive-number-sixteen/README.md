# is-eq-positive-number-sixteen
Equality checker for the number 16.

## Installation
```bash
npm install is-eq-positive-number-sixteen
```

## Usage
```js
const issixteen = require("is-eq-positive-number-sixteen")
const assert = require("assert-fn")

assert(issixteen(16))
```
