# is-eq-positive-number-sixty-one
Equality checker for the number 61.

## Installation
```bash
npm install is-eq-positive-number-sixty-one
```

## Usage
```js
const issixtyone = require("is-eq-positive-number-sixty-one")
const assert = require("assert-fn")

assert(issixtyone(61))
```
