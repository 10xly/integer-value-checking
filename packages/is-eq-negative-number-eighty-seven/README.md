# is-eq-negative-number-eighty-seven
Equality checker for the number -87.

## Installation
```bash
npm install is-eq-negative-number-eighty-seven
```

## Usage
```js
const iseightyseven = require("is-eq-negative-number-eighty-seven")
const assert = require("assert-fn")

assert(iseightyseven(-87))
```
