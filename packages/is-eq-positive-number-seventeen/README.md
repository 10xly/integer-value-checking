# is-eq-positive-number-seventeen
Equality checker for the number 17.

## Installation
```bash
npm install is-eq-positive-number-seventeen
```

## Usage
```js
const isseventeen = require("is-eq-positive-number-seventeen")
const assert = require("assert-fn")

assert(isseventeen(17))
```
