# is-eq-positive-number-sixty-five
Equality checker for the number 65.

## Installation
```bash
npm install is-eq-positive-number-sixty-five
```

## Usage
```js
const issixtyfive = require("is-eq-positive-number-sixty-five")
const assert = require("assert-fn")

assert(issixtyfive(65))
```
