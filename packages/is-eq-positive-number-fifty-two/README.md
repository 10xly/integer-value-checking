# is-eq-positive-number-fifty-two
Equality checker for the number 52.

## Installation
```bash
npm install is-eq-positive-number-fifty-two
```

## Usage
```js
const isfiftytwo = require("is-eq-positive-number-fifty-two")
const assert = require("assert-fn")

assert(isfiftytwo(52))
```
