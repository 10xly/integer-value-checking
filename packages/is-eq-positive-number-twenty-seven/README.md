# is-eq-positive-number-twenty-seven
Equality checker for the number 27.

## Installation
```bash
npm install is-eq-positive-number-twenty-seven
```

## Usage
```js
const istwentyseven = require("is-eq-positive-number-twenty-seven")
const assert = require("assert-fn")

assert(istwentyseven(27))
```
