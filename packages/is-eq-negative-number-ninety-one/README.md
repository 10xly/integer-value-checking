# is-eq-negative-number-ninety-one
Equality checker for the number -91.

## Installation
```bash
npm install is-eq-negative-number-ninety-one
```

## Usage
```js
const isninetyone = require("is-eq-negative-number-ninety-one")
const assert = require("assert-fn")

assert(isninetyone(-91))
```
