# is-eq-negative-number-eighty-four
Equality checker for the number -84.

## Installation
```bash
npm install is-eq-negative-number-eighty-four
```

## Usage
```js
const iseightyfour = require("is-eq-negative-number-eighty-four")
const assert = require("assert-fn")

assert(iseightyfour(-84))
```
