# is-eq-negative-number-sixty-two
Equality checker for the number -62.

## Installation
```bash
npm install is-eq-negative-number-sixty-two
```

## Usage
```js
const issixtytwo = require("is-eq-negative-number-sixty-two")
const assert = require("assert-fn")

assert(issixtytwo(-62))
```
