# is-eq-negative-number-fourteen
Equality checker for the number -14.

## Installation
```bash
npm install is-eq-negative-number-fourteen
```

## Usage
```js
const isfourteen = require("is-eq-negative-number-fourteen")
const assert = require("assert-fn")

assert(isfourteen(-14))
```
