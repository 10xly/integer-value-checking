# is-eq-positive-number-twenty-four
Equality checker for the number 24.

## Installation
```bash
npm install is-eq-positive-number-twenty-four
```

## Usage
```js
const istwentyfour = require("is-eq-positive-number-twenty-four")
const assert = require("assert-fn")

assert(istwentyfour(24))
```
