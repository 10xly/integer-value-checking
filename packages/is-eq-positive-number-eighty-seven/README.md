# is-eq-positive-number-eighty-seven
Equality checker for the number 87.

## Installation
```bash
npm install is-eq-positive-number-eighty-seven
```

## Usage
```js
const iseightyseven = require("is-eq-positive-number-eighty-seven")
const assert = require("assert-fn")

assert(iseightyseven(87))
```
