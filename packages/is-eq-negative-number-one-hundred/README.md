# is-eq-negative-number-one-hundred
Equality checker for the number -100.

## Installation
```bash
npm install is-eq-negative-number-one-hundred
```

## Usage
```js
const isonehundred = require("is-eq-negative-number-one-hundred")
const assert = require("assert-fn")

assert(isonehundred(-100))
```
