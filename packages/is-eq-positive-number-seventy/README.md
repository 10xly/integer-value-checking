# is-eq-positive-number-seventy
Equality checker for the number 70.

## Installation
```bash
npm install is-eq-positive-number-seventy
```

## Usage
```js
const isseventy = require("is-eq-positive-number-seventy")
const assert = require("assert-fn")

assert(isseventy(70))
```
