# is-eq-negative-number-forty-four
Equality checker for the number -44.

## Installation
```bash
npm install is-eq-negative-number-forty-four
```

## Usage
```js
const isfortyfour = require("is-eq-negative-number-forty-four")
const assert = require("assert-fn")

assert(isfortyfour(-44))
```
