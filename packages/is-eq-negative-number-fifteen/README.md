# is-eq-negative-number-fifteen
Equality checker for the number -15.

## Installation
```bash
npm install is-eq-negative-number-fifteen
```

## Usage
```js
const isfifteen = require("is-eq-negative-number-fifteen")
const assert = require("assert-fn")

assert(isfifteen(-15))
```
