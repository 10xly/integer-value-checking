# is-eq-negative-number-twenty-six
Equality checker for the number -26.

## Installation
```bash
npm install is-eq-negative-number-twenty-six
```

## Usage
```js
const istwentysix = require("is-eq-negative-number-twenty-six")
const assert = require("assert-fn")

assert(istwentysix(-26))
```
