# is-eq-positive-number-forty-five
Equality checker for the number 45.

## Installation
```bash
npm install is-eq-positive-number-forty-five
```

## Usage
```js
const isfortyfive = require("is-eq-positive-number-forty-five")
const assert = require("assert-fn")

assert(isfortyfive(45))
```
