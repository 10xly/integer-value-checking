# is-eq-positive-number-forty-nine
Equality checker for the number 49.

## Installation
```bash
npm install is-eq-positive-number-forty-nine
```

## Usage
```js
const isfortynine = require("is-eq-positive-number-forty-nine")
const assert = require("assert-fn")

assert(isfortynine(49))
```
