# is-eq-negative-number-twenty-four
Equality checker for the number -24.

## Installation
```bash
npm install is-eq-negative-number-twenty-four
```

## Usage
```js
const istwentyfour = require("is-eq-negative-number-twenty-four")
const assert = require("assert-fn")

assert(istwentyfour(-24))
```
