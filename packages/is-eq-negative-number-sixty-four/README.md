# is-eq-negative-number-sixty-four
Equality checker for the number -64.

## Installation
```bash
npm install is-eq-negative-number-sixty-four
```

## Usage
```js
const issixtyfour = require("is-eq-negative-number-sixty-four")
const assert = require("assert-fn")

assert(issixtyfour(-64))
```
