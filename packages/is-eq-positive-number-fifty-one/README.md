# is-eq-positive-number-fifty-one
Equality checker for the number 51.

## Installation
```bash
npm install is-eq-positive-number-fifty-one
```

## Usage
```js
const isfiftyone = require("is-eq-positive-number-fifty-one")
const assert = require("assert-fn")

assert(isfiftyone(51))
```
