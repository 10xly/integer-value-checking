# is-eq-negative-number-thirteen
Equality checker for the number -13.

## Installation
```bash
npm install is-eq-negative-number-thirteen
```

## Usage
```js
const isthirteen = require("is-eq-negative-number-thirteen")
const assert = require("assert-fn")

assert(isthirteen(-13))
```
