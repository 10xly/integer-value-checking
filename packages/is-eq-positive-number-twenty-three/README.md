# is-eq-positive-number-twenty-three
Equality checker for the number 23.

## Installation
```bash
npm install is-eq-positive-number-twenty-three
```

## Usage
```js
const istwentythree = require("is-eq-positive-number-twenty-three")
const assert = require("assert-fn")

assert(istwentythree(23))
```
