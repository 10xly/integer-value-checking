# is-eq-positive-number-seventy-six
Equality checker for the number 76.

## Installation
```bash
npm install is-eq-positive-number-seventy-six
```

## Usage
```js
const isseventysix = require("is-eq-positive-number-seventy-six")
const assert = require("assert-fn")

assert(isseventysix(76))
```
