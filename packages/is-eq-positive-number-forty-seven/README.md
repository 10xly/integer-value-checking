# is-eq-positive-number-forty-seven
Equality checker for the number 47.

## Installation
```bash
npm install is-eq-positive-number-forty-seven
```

## Usage
```js
const isfortyseven = require("is-eq-positive-number-forty-seven")
const assert = require("assert-fn")

assert(isfortyseven(47))
```
