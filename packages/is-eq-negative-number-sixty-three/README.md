# is-eq-negative-number-sixty-three
Equality checker for the number -63.

## Installation
```bash
npm install is-eq-negative-number-sixty-three
```

## Usage
```js
const issixtythree = require("is-eq-negative-number-sixty-three")
const assert = require("assert-fn")

assert(issixtythree(-63))
```
