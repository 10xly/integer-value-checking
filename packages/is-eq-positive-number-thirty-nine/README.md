# is-eq-positive-number-thirty-nine
Equality checker for the number 39.

## Installation
```bash
npm install is-eq-positive-number-thirty-nine
```

## Usage
```js
const isthirtynine = require("is-eq-positive-number-thirty-nine")
const assert = require("assert-fn")

assert(isthirtynine(39))
```
