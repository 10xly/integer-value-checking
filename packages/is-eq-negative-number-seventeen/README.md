# is-eq-negative-number-seventeen
Equality checker for the number -17.

## Installation
```bash
npm install is-eq-negative-number-seventeen
```

## Usage
```js
const isseventeen = require("is-eq-negative-number-seventeen")
const assert = require("assert-fn")

assert(isseventeen(-17))
```
