# is-eq-positive-number-sixty-nine
Equality checker for the number 69.

## Installation
```bash
npm install is-eq-positive-number-sixty-nine
```

## Usage
```js
const issixtynine = require("is-eq-positive-number-sixty-nine")
const assert = require("assert-fn")

assert(issixtynine(69))
```
