# is-eq-negative-number-seventy-two
Equality checker for the number -72.

## Installation
```bash
npm install is-eq-negative-number-seventy-two
```

## Usage
```js
const isseventytwo = require("is-eq-negative-number-seventy-two")
const assert = require("assert-fn")

assert(isseventytwo(-72))
```
