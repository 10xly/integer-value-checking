# is-eq-negative-number-eighty-one
Equality checker for the number -81.

## Installation
```bash
npm install is-eq-negative-number-eighty-one
```

## Usage
```js
const iseightyone = require("is-eq-negative-number-eighty-one")
const assert = require("assert-fn")

assert(iseightyone(-81))
```
