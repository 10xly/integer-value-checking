# is-eq-negative-number-fifty-four
Equality checker for the number -54.

## Installation
```bash
npm install is-eq-negative-number-fifty-four
```

## Usage
```js
const isfiftyfour = require("is-eq-negative-number-fifty-four")
const assert = require("assert-fn")

assert(isfiftyfour(-54))
```
