# is-eq-negative-number-forty-seven
Equality checker for the number -47.

## Installation
```bash
npm install is-eq-negative-number-forty-seven
```

## Usage
```js
const isfortyseven = require("is-eq-negative-number-forty-seven")
const assert = require("assert-fn")

assert(isfortyseven(-47))
```
