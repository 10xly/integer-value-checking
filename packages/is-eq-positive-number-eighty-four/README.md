# is-eq-positive-number-eighty-four
Equality checker for the number 84.

## Installation
```bash
npm install is-eq-positive-number-eighty-four
```

## Usage
```js
const iseightyfour = require("is-eq-positive-number-eighty-four")
const assert = require("assert-fn")

assert(iseightyfour(84))
```
