# is-eq-negative-number-thirty-two
Equality checker for the number -32.

## Installation
```bash
npm install is-eq-negative-number-thirty-two
```

## Usage
```js
const isthirtytwo = require("is-eq-negative-number-thirty-two")
const assert = require("assert-fn")

assert(isthirtytwo(-32))
```
