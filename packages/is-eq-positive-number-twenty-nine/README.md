# is-eq-positive-number-twenty-nine
Equality checker for the number 29.

## Installation
```bash
npm install is-eq-positive-number-twenty-nine
```

## Usage
```js
const istwentynine = require("is-eq-positive-number-twenty-nine")
const assert = require("assert-fn")

assert(istwentynine(29))
```
