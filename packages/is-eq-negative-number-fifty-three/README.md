# is-eq-negative-number-fifty-three
Equality checker for the number -53.

## Installation
```bash
npm install is-eq-negative-number-fifty-three
```

## Usage
```js
const isfiftythree = require("is-eq-negative-number-fifty-three")
const assert = require("assert-fn")

assert(isfiftythree(-53))
```
