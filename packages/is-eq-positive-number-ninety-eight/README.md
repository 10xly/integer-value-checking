# is-eq-positive-number-ninety-eight
Equality checker for the number 98.

## Installation
```bash
npm install is-eq-positive-number-ninety-eight
```

## Usage
```js
const isninetyeight = require("is-eq-positive-number-ninety-eight")
const assert = require("assert-fn")

assert(isninetyeight(98))
```
