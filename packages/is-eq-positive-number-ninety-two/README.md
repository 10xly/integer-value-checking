# is-eq-positive-number-ninety-two
Equality checker for the number 92.

## Installation
```bash
npm install is-eq-positive-number-ninety-two
```

## Usage
```js
const isninetytwo = require("is-eq-positive-number-ninety-two")
const assert = require("assert-fn")

assert(isninetytwo(92))
```
