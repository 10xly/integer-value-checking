# is-eq-positive-number-eighty-eight
Equality checker for the number 88.

## Installation
```bash
npm install is-eq-positive-number-eighty-eight
```

## Usage
```js
const iseightyeight = require("is-eq-positive-number-eighty-eight")
const assert = require("assert-fn")

assert(iseightyeight(88))
```
