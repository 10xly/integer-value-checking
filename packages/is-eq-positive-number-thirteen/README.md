# is-eq-positive-number-thirteen
Equality checker for the number 13.

## Installation
```bash
npm install is-eq-positive-number-thirteen
```

## Usage
```js
const isthirteen = require("is-eq-positive-number-thirteen")
const assert = require("assert-fn")

assert(isthirteen(13))
```
