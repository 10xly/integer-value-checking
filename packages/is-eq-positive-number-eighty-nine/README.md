# is-eq-positive-number-eighty-nine
Equality checker for the number 89.

## Installation
```bash
npm install is-eq-positive-number-eighty-nine
```

## Usage
```js
const iseightynine = require("is-eq-positive-number-eighty-nine")
const assert = require("assert-fn")

assert(iseightynine(89))
```
