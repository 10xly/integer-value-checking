# is-eq-negative-number-sixty-seven
Equality checker for the number -67.

## Installation
```bash
npm install is-eq-negative-number-sixty-seven
```

## Usage
```js
const issixtyseven = require("is-eq-negative-number-sixty-seven")
const assert = require("assert-fn")

assert(issixtyseven(-67))
```
