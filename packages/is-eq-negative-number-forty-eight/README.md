# is-eq-negative-number-forty-eight
Equality checker for the number -48.

## Installation
```bash
npm install is-eq-negative-number-forty-eight
```

## Usage
```js
const isfortyeight = require("is-eq-negative-number-forty-eight")
const assert = require("assert-fn")

assert(isfortyeight(-48))
```
