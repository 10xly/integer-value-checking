# is-eq-negative-number-fifty-nine
Equality checker for the number -59.

## Installation
```bash
npm install is-eq-negative-number-fifty-nine
```

## Usage
```js
const isfiftynine = require("is-eq-negative-number-fifty-nine")
const assert = require("assert-fn")

assert(isfiftynine(-59))
```
