# is-eq-positive-number-fifty-four
Equality checker for the number 54.

## Installation
```bash
npm install is-eq-positive-number-fifty-four
```

## Usage
```js
const isfiftyfour = require("is-eq-positive-number-fifty-four")
const assert = require("assert-fn")

assert(isfiftyfour(54))
```
