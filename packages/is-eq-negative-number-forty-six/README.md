# is-eq-negative-number-forty-six
Equality checker for the number -46.

## Installation
```bash
npm install is-eq-negative-number-forty-six
```

## Usage
```js
const isfortysix = require("is-eq-negative-number-forty-six")
const assert = require("assert-fn")

assert(isfortysix(-46))
```
