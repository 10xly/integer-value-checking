# is-eq-positive-number-twenty-one
Equality checker for the number 21.

## Installation
```bash
npm install is-eq-positive-number-twenty-one
```

## Usage
```js
const istwentyone = require("is-eq-positive-number-twenty-one")
const assert = require("assert-fn")

assert(istwentyone(21))
```
