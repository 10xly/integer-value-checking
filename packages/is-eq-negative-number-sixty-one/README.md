# is-eq-negative-number-sixty-one
Equality checker for the number -61.

## Installation
```bash
npm install is-eq-negative-number-sixty-one
```

## Usage
```js
const issixtyone = require("is-eq-negative-number-sixty-one")
const assert = require("assert-fn")

assert(issixtyone(-61))
```
