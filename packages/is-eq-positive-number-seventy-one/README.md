# is-eq-positive-number-seventy-one
Equality checker for the number 71.

## Installation
```bash
npm install is-eq-positive-number-seventy-one
```

## Usage
```js
const isseventyone = require("is-eq-positive-number-seventy-one")
const assert = require("assert-fn")

assert(isseventyone(71))
```
