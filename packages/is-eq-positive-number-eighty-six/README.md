# is-eq-positive-number-eighty-six
Equality checker for the number 86.

## Installation
```bash
npm install is-eq-positive-number-eighty-six
```

## Usage
```js
const iseightysix = require("is-eq-positive-number-eighty-six")
const assert = require("assert-fn")

assert(iseightysix(86))
```
