# is-eq-positive-number-forty-eight
Equality checker for the number 48.

## Installation
```bash
npm install is-eq-positive-number-forty-eight
```

## Usage
```js
const isfortyeight = require("is-eq-positive-number-forty-eight")
const assert = require("assert-fn")

assert(isfortyeight(48))
```
