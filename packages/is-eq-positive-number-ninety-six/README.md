# is-eq-positive-number-ninety-six
Equality checker for the number 96.

## Installation
```bash
npm install is-eq-positive-number-ninety-six
```

## Usage
```js
const isninetysix = require("is-eq-positive-number-ninety-six")
const assert = require("assert-fn")

assert(isninetysix(96))
```
