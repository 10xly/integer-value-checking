# is-eq-positive-number-ninety-one
Equality checker for the number 91.

## Installation
```bash
npm install is-eq-positive-number-ninety-one
```

## Usage
```js
const isninetyone = require("is-eq-positive-number-ninety-one")
const assert = require("assert-fn")

assert(isninetyone(91))
```
