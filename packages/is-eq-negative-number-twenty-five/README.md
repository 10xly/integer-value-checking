# is-eq-negative-number-twenty-five
Equality checker for the number -25.

## Installation
```bash
npm install is-eq-negative-number-twenty-five
```

## Usage
```js
const istwentyfive = require("is-eq-negative-number-twenty-five")
const assert = require("assert-fn")

assert(istwentyfive(-25))
```
