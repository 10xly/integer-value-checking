# is-eq-positive-number-fifty-seven
Equality checker for the number 57.

## Installation
```bash
npm install is-eq-positive-number-fifty-seven
```

## Usage
```js
const isfiftyseven = require("is-eq-positive-number-fifty-seven")
const assert = require("assert-fn")

assert(isfiftyseven(57))
```
