# is-eq-negative-number-seventy-nine
Equality checker for the number -79.

## Installation
```bash
npm install is-eq-negative-number-seventy-nine
```

## Usage
```js
const isseventynine = require("is-eq-negative-number-seventy-nine")
const assert = require("assert-fn")

assert(isseventynine(-79))
```
