# is-eq-negative-number-seventy-seven
Equality checker for the number -77.

## Installation
```bash
npm install is-eq-negative-number-seventy-seven
```

## Usage
```js
const isseventyseven = require("is-eq-negative-number-seventy-seven")
const assert = require("assert-fn")

assert(isseventyseven(-77))
```
