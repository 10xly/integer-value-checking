# is-eq-negative-number-fifty-five
Equality checker for the number -55.

## Installation
```bash
npm install is-eq-negative-number-fifty-five
```

## Usage
```js
const isfiftyfive = require("is-eq-negative-number-fifty-five")
const assert = require("assert-fn")

assert(isfiftyfive(-55))
```
